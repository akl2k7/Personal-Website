About me

Hello, my name is Andrew Laino and I am an aspiring front-end web developer with an Associate's Degree in Computer Science. I know HTML, CSS and JavaScript quite well and I also know how to use the jQuery library effectively. In addition, I also know how to spruce pages up with the Bootstrap framework, giving them a great look in a matter of minutes. If you want my services, feel free to contact me using the info on my contact page.

Along with all of this, I have a number of hobbies. For instance, I learned Spanish as an adult and I speak with a near native accent and I've toyed with a number of other languages including Russian and Latin. I have also looked into game programming, though I have been playing a lot more videogames, particularly western roleplaying games such as Skyrim and Fallout. Lately, I have been learning how to be a game master for a tabeltop roleplaying game as well, which provides a fun excuse to have friends over. Along with all of that, I enjoy drawing and writing stories. Plus, I'm a total Star Wars and Star Trek geek.
